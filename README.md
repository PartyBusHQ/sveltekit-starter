# Getting Started

```
git clone https://github.com/diamondalltheway/svelte-starter-kit.git
cd svelte-starter-kit
npm i
npm run dev
```

## Basic Info

This starter kit uses Sveltekit (adapter-auto).

### CSS / UI

- [x] Tailwind CSS
- [x] DaisyUI

### Database

- [x] Supabase
- [x] MongoDB

### Services

- [x] Algolia
- [x] Contentful

### Testing

- [x] Playwright
- [x] Vitest
