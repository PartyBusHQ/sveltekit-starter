# Getting Started

```
git clone https://github.com/diamondalltheway/svelte-starter-kit.git
cd svelte-starter-kit
npm i
npm run dev
```

### ENV VARS

```
MongoDB_URI
Supabase_Anon_key
Supabase_URL
DATABASE_URL
DIRECT_URL
```
